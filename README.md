# An analysis for matchmaking system

> **Note**: This project might take months to a year as I am preparing for the National Postgraduate Entrance Examination and looking for a job (if I mess the exam up). So please be patient with me, thanks!

## Overview

This repository aims for studying the stability of matmaking system in many competitive games, especially in Overwatch. For many reasons, smurfing and boosting plague many games, and they are impossible to completely eradicate. Therefore, evaluating the negative impact on the matchmaking system is worthwhile.

## Roadmap

- Phase 0: Figure out the mathematical theory framework
- Phase 1: Simulate with Monte Carlo method, and submit a decent report with data visualization
